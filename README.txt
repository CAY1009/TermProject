A simple console-based Library Management System implemented in Java. This system enables users to manage books, borrowers, and transactions, including borrowing and returning books. Data is saved and loaded from text files (`books.txt`, `borrowers.txt`).

---
Book Management

- Add, edit, delete, and list books
- Each book has the following attributes:
  - ID
  - Title
  - Author
  - Genre
  - Availability (whether the book is borrowed or available)

---
Borrower Management

- Register, update, delete, and list borrowers
- Each borrower has the following attributes:
  - ID
  - Name
  - Contact details

---
Borrow & Return Books

- Borrow available books and mark them as borrowed
- Return borrowed books and mark them as available

---
Search & Filter

- Search books by title, author, or genre
- Filter books by availability status (Available/Borrowed)

---
File Handling

- Data is stored in text files:
  - `books.txt` stores books with details like ID, title, author, genre, and availability
  - `borrowers.txt` stores borrower details like ID, name, and contact
